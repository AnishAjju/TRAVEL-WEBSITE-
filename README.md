# Travel Insurance Landing Page

This is a responsive travel insurance landing page built with pure HTML5, CSS3, and JavaScript.

## How to Run

1. Open the project folder.
2. Double-click `index.html` to open it in your default web browser.
3. Alternatively, you can serve it using a local development server like Live Server in VS Code or `python -m http.server`.

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile.
- **Interactive Form**:
    - Tab switching between Single and Multi Trip.
    - Tag management for destinations.
    - Dynamic traveler age inputs.
- **Performance**:
    - Optimized asset loading.
    - semantic HTML.
    - accessible contrast ratios.

## Technologies

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript (Vanilla)
